# EtherChannel Configuration (LACP, PAgP & Layer 3)

Lab Description
In this lab, I configured different types of EtherChannel connections between access and distribution switches

Objectives
1. Configure Layer 2 EtherChannel between ASW1 and DSW1 using LACP and set it as a trunk
2. Configure Layer 2 EtherChannel between ASW2 and DSW2 using PAgP and set it as a trunk
3. Configure Layer 3 EtherChannel between DSW1 and DSW2 using static EtherChannel
4. Configure routing to allow PCs to reach SRV1 by using SVI on Layer 3 switch
5. Identify the default EtherChannel load-balancing method
6. Configure load-balancing based on source and destination IP addresses

Devices Used
1. ASW1 – Access Switch 1
2. ASW2 – Access Switch 2
3. DSW1 – Distribution Switch 1
4. DSW2 – Distribution Switch 2
5. PCs
6. SRV1
